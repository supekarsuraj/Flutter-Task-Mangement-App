# Flutter-Task-Mangement-App

Features 
 Add Tasks Easily 📝 
 Mark Tasks as Completed ✅ 
 Delete Tasks 🗑️ 
 Light & Dark Mode Support 🌗 
 State Management with BLoC ⚡
